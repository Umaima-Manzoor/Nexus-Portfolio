# Nexus – Creative Portfolio with Dynamic Hero Slider

A modern, responsive portfolio website featuring a full-width animated hero slider, dynamic content from JSON, services section, project gallery, testimonials, and scroll-triggered animations.

## 🚀 Live Demo
[View on Vercel](https://your-vercel-url.vercel.app)  *(replace after deploy)*

## 📋 Task Requirements Fulfilled
- ✅ Full-width hero slider with auto-rotate (4.5s)
- ✅ Manual navigation (prev/next arrows + dots)
- ✅ Each slide: image, title, description, CTA button
- ✅ Smooth fade/slide transitions + hover effects
- ✅ Fully responsive (desktop, tablet, mobile)
- ✅ Slide data stored in JSON (JavaScript object)
- ✅ Semantic HTML5 & accessibility (alt text, keyboard nav)
- ✅ Optional: mouse parallax effect on floating shapes

## 🛠️ Technologies Used
- HTML5
- CSS3 (variables, grid, flexbox, keyframes)
- Vanilla JavaScript (no frameworks)
- Font Awesome 6
- Google Fonts (Space Grotesk, Syne)

## 🧩 Key Features
- **Dynamic Hero Slider** – built from JSON array, touch/swipe support, progress bar
- **Services & Projects** – dynamically rendered cards
- **Counter Animation** – stats numbers animate on scroll
- **Scroll Reveal** – elements fade in as you scroll
- **Toast Notifications** – user feedback on button clicks
- **Floating Shapes** – subtle background parallax on mouse move
